             taskkill /PID [PID] /F
    Or use: npm run dev -- -p 3001

Problem: "Module not found" errors
Solution: 
    rm -rf node_modules package-lock.json
    npm install

Problem: Styles not loading
Solution:
    1. Hard refresh browser (Ctrl + F5)
    2. Check terminal for errors
    3. Restart dev server
